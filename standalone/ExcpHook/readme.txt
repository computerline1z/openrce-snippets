ExcpHook is an open source (see license.txt) Exception Monitor for
Windows made by Gynvael Coldwind (Team Vexillium).

Currently supported Windows versions: XP SP2

Please note that this is ALPHA version.

The compiled version of this tool is available on http://vexillium.org

Contact with the author:
e-mail   : gynvael@vexillium.org
www      : http://gynvael.vexillium.org

Bug reports / feature requests / comments are welcomed! ;>

Compilation: use DDK to compile the driver, and MinGW G++
             to compile the EXE
